# 延迟渲染管线（当前实现）

## 已实现要点

当前延迟渲染在 `RenderEngine` 中已接入：

- 渲染模式切换：`Forward / Deferred`
- `DeferredGBuffer` 几何缓冲
- Geometry Pass + Lighting Pass
- Forward overlay（天空盒、透明对象）
- Debug View（Final / Position / Normal / Albedo / Roughness / Metallic）

主要实现位置：

- `JGL_Engine/source/engine/render_engine.h`
- `JGL_Engine/source/engine/render_engine.cpp`
- `JGL_Engine/source/render/deferred_gbuffer.h`
- `JGL_Engine/source/render/deferred_gbuffer.cpp`
- `JGL_Engine/shaders/deferred_gbuffer_*.shader`
- `JGL_Engine/shaders/deferred_lighting_*.shader`

## 当前限制

- 延迟路径可用性受材质贴图完整性影响（缺少必要 PBR 贴图时会回退）
- 透明对象仍走 forward overlay
- 更多后处理（如完整 Bloom/SSAO/SSR）未在延迟链路中完整集成

## 说明

旧版“延迟渲染设计方案”文档以改造计划为主，包含较多历史内容（如 `SceneView` 级改造路径）。
